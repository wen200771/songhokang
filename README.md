-   快速啟動

    -   PHP 伺服器：`php -S localhost:8000`
    -   Windows 批次：`scripts/start_server.bat`
    -   前台入口：`http://localhost:8000/`
    -   後台入口：`/admin/`, `/vendor/`, `/user/`
